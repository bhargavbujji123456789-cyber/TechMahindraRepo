# Parabank Login Automation

## Prerequisites
- .NET 6 SDK or later installed
- Chrome browser installed
- ChromeDriver matching your Chrome version (or use WebDriverManager)

## Setup
- Clone the repository
- Open the solution in Visual Studio or your preferred IDE
- Restore NuGet packages

## Running Tests
- Run tests using NUnit Test Explorer or command line:
  `dotnet test`

## Notes
- Tests use Selenium WebDriver with ChromeDriver
- Tests cover valid login, invalid login, and empty field validations
